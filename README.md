FAEDED — Desktop Music Player (Java, JavaFX, JFoenix, SQLite, Maven)
- Built a JavaFX desktop music player using the MVC pattern, with JFoenix for Material Design UI and SQLite for persistent song library storage
- Implemented core playback (play/pause/seek/shuffle/repeat/volume) via JavaFX MediaPlayer, with automatic metadata/album-art extraction
- Designed a searchable library drawer with custom list-cell rendering and a frameless, custom-styled UI
